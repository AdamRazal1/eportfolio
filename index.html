Scrape Master — Carousell HPC Pipeline

Cloud Data Engineering Apr 2026

Azure End-to-End Data Pipeline

A full Azure pipeline that ingests customer and sales data from an on-premises SQL database into Data Lake Storage Gen2 via Azure Data Factory, transforms it through a Bronze–Silver–Gold medallion architecture in Databricks (PySpark), and loads it into Synapse Analytics. Time-Series Forecasting Dec 2025

UMDAC Datathon '25 — Cash Flow Analytics
